# 职想 - 职业测评网站

这是一个为“职想”校园招聘微信小程序提供的外部增值服务，旨在帮助学生进行职业性格和职业兴趣测评。本网站是一个纯静态页面，部署在 GitHub Pages 上，并通过小程序进行跳转访问。

## 功能特色

- **职业性格测试**：基于大五人格理论，帮助用户了解自身性格特质。
- **职业兴趣测试**：基于霍兰德职业兴趣理论，帮助用户探索适合的职业领域。
- **简约设计**：采用绿色为主色调的简约风格，提供良好的用户体验。
- **分享卡片**：测试完成后可生成精美的分享卡片，方便用户分享测试结果。
- **烟花特效**：完成测试时会触发绚丽的烟花爆炸效果，增添趣味性和仪式感。
- **鲜花特效**：完成测试后，多朵鲜花从屏幕上方缓缓飘落，营造温暖欢乐的氛围。
- **光点绽放**：光点汇聚绽放效果，强化成功完成的反馈感。

## 技术栈

- HTML5
- CSS3
- JavaScript (原生JS，无框架)
- html2canvas (用于生成分享卡片)

## 文件结构

- `index.html`: 网站首页，提供两种测试的入口。
- `style.css`: 全局样式文件，定义网站的视觉风格。
- `personality_test.html`: 职业性格测试页面。
- `interest_test.html`: 职业兴趣测试页面。
- `script.js`: 核心JavaScript文件，包含测试逻辑、结果计算、结果展示和分享卡片生成功能。

## 部署方法 (GitHub Pages)

1. **克隆仓库**：将本项目克隆到你的本地计算机。
   ```bash
   git clone <你的仓库URL>
   cd career_tests_website
   ```
2. **创建GitHub仓库**：在 GitHub 上创建一个新的公共仓库。
3. **推送到GitHub**：将本地代码推送到你创建的 GitHub 仓库。
   ```bash
   git remote add origin <你的GitHub仓库URL>
   git branch -M main
   git push -u origin main
   ```
4. **配置GitHub Pages**：
   - 进入你的 GitHub 仓库页面。
   - 点击 `Settings` (设置)。
   - 在左侧导航栏中选择 `Pages`。
   - 在 `Source` (来源) 部分，选择 `Branch: main` (或你使用的分支)，并选择 `/ (root)` 文件夹。
   - 点击 `Save` (保存)。
5. **访问网站**：等待几分钟，GitHub Pages 将会自动部署你的网站。部署成功后，你会在 `Pages` 设置页面看到你的网站 URL (通常是 `https://<你的用户名>.github.io/<你的仓库名>/`)。

## 如何使用

1. 访问部署后的网站首页。
2. 选择“职业性格测试”或“职业兴趣测试”开始答题。
3. 完成所有题目后，系统将显示你的测试结果。
4. 点击“生成分享卡片”按钮，即可将结果保存为图片。

## 集成到微信小程序

小程序可以直接通过 `webview` 组件或 `wx.navigateTo` 等方式跳转到本网站的 URL。测试结果无需回传至小程序，学生可手动将分享卡片保存后上传至小程序或填写相关信息。
